# 张昌蒲计算器 zcp-calculator

## 简介

对于三国杀十周年区武将张昌蒲的快速计算APP，采用vue+composition-api+vuex+element-ui+typescript构造web端，利用uni-app构建手机安卓端。

## 源代码安装dev
```
npm install
```

### 运行
```
npm run serve
```

### 打包
```
npm run build
```
